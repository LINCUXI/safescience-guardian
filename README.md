# 🛡️ SafeScience 守護者 — AI 實驗室安全管理系統

一套專為理化實驗課設計的 AI 輔助安全評分工具，幫助老師即時掌握學生行為並管理各組安全積分。

---

## 🌐 線上展示

| 用途 | 連結 |
|------|------|
| 🖥️ 主畫面（投影至大螢幕） | [開啟主畫面](https://lincuxi.github.io/safescience-guardian/) |
| 📱 行動遙控器（老師手機操作） | [開啟遙控器](https://lincuxi.github.io/safescience-guardian/remote.html) |

---

## ✨ 功能特色

- 🤖 **AI 行為偵測**：透過攝影機即時追蹤人臉移動，自動標記疑似嬉戲行為
- 📊 **即時積分榜**：六組同時顯示，動態更新 MVP 安全模範組
- 📱 **行動遙控器**：老師用手機即時加扣分，支援觸覺震動回饋
- 📥 **匯出紀錄**：一鍵匯出 CSV 實驗紀錄，方便課後檢討

---

## 🚀 使用方式

1. 在電腦開啟**主畫面**，投影到教室大螢幕
2. 用手機掃描 QR Code 或直接開啟**遙控器網址**
3. 點選「啟動 AI 守護之眼」，允許攝影機權限
4. 選擇組別後即可開始加扣分

> ⚠️ 主畫面與遙控器需在**同一台電腦的同一瀏覽器**中開啟，同步功能才有效果。

---

## 🛠️ 技術架構

- 純前端實作，無需安裝任何套件
- `BroadcastChannel API` — 主畫面與遙控器即時同步
- `tracking.js` — 瀏覽器端人臉偵測
- `Tailwind CSS` — 響應式介面設計
- `Web Audio API` — 加扣分音效回饋
